JavaWEB-（二十二）BootStrap-组件(三)

本文最后更新于：December 3, 2021 pm

Bootstrap 是最受欢迎的 HTML、CSS 和 JS 框架，用于开发响应式布局、移动设备优先的 WEB 项目。Bootstrap是基于HTML、CSS、JavaScript 开发的简洁、直观、强悍的前端开发框架，使得 Web 开发更加快捷。提供了优雅的HTML和CSS规范。

2.标签

标签可用于计数、提示或页面上其他的标记显示。使用 .label 来显示标签。

可以使用修饰的 label-default、label-primary、label-success、label-info、label-warning、label-danger 来改变标签的外观。

3.徽章

徽章（Badges）主要用于突出显示新的或未读的项。徽章与标签相似，主要的区别在于徽章的边角更加圆滑。给链接、导航等元素嵌套 <span class="badge"> 元素，可以很醒目的展示新的或未读的信息条目。

如果没有新的或未读的信息条目，也就是说不包含任何内容，徽章组件能够自动隐藏（通过CSS的 :empty 选择符实现) 。

3.1 导航元素的激活状态

可以在激活状态的胶囊式导航和列表导航中放置徽章。通过使用 来激活链接。让胶囊式导航内处于激活状态的元素所包含的徽章展示相匹配的样式。

4.缩略图

通过缩略图组件扩展 Bootstrap 的 栅格系统，可以很容易地以网格的方式展示图像、视频、文本等内容。

4.1 默认样式

缩略图的默认设计仅需最少的标签就能展示带链接的图片。

使用 Bootstrap 创建缩略图的步骤如下：

  • 在图像周围添加带有 .thumbnail 的 标签。
  • 这会添加四个像素的内边距（padding）和一个灰色的边框。
  • 当鼠标悬停在图像上时，会动画显示出图像的轮廓。

4.2 自定义内容

添加一点点额外的标签，就可以把任何类型的 HTML 内容，例如标题、段落或按钮，加入缩略图组件内。
