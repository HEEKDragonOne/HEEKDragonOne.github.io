ClashA教程 | 安卓

本文最后更新于：December 3, 2021 pm

此软件已停止更新，仅提供软件功能BUG修复。
并不影响当前版本的使用。详情请看文末说明。

Clash 是一个使用 Go 语言编写，基于规则的跨平台代理软件核心程序。
ClashA 是目前在 Andorid 上表现较为出色的图形化 Clash 分支。
Github 项目地址： https://github.com/ccg2018/ClashA
特点：

  • 可随时切换代理模式及节点
  • 支持节点批量延迟测试
  • 通过托管链接一键配置
  • 规则命中分析
  • 日志输出

4. 快速配置 ClashA

首先你需要有一个 Clash 订阅链接（ Windows/Android/MacOS 通用） 或 YML/YAML 格式的配置文件。
配置文件的导入方式为点击软件右上角的 Import Config 导入即可。本文仅针对订阅作出说明。

支持 Clash 的机场一般都会直接提供 Clash 订阅链接。如果你是机场用户，请先进入机场的主控面板获取订阅链接。

下图仅为示例，请以实际情况为准：

复制好了 Clash 订阅之后打开 ClashA 应用程序。

点击右上角三个点的按钮。选择 Download Config 。

粘贴已事先复制好的 Clash 订阅链接。

等待若干秒，你将看到下载成功的配置文件卡片。

右上角这里是刷新按钮。机场服务器信息可能会不定时更新，若出现大面积节点超时现象，可尝试此项目。

点击右上角 Dashboard， 可进入策略组面板。

默认地址一般是 127.0.0.1 ，默认端口为 7892 ，默认不留密钥。
具体情况视具体的配置文件写法而定。

不同的机场使用不同的分流规则、策略、策略组等配置，，以下画面仅供参考。请以自己的实际情况为准。

在 代理 界面选择一个节点之后即可关掉窗口，点击首页右下角的猫咪（代理开关）即可创建代理，出国留学。

5. 常见错误

如果遇到上图提示，可能的原因有：

  • 配置有问题
  • Country.db 有问题，右上角子菜单fix Country.mmdb
  • DNS server 为空，请检查设置

6. 其它

点击左上角的三道杠标志，查看更多设置。

点击 Global Settings

不建议打开 IPv 6 的支持。很多网友反映在启用 IPv 6 之后脸书和IG图片无法加载，可能是因为安卓机优先使用 IPv 6,但目前国内 IPv 6 的环境一言难尽……

如果你不明白以下选项的含义，请不要更改，保持默认就好。

7. 关于 Clash for Android

日前，ClashA 开发者 @ccg2018 宣布 ClashA 将停止更新，并转由新的开发者 @Kr328 进行开发维护，新的项目命名为 Clash for Android。

系统要求

  • Android 7.0 及以上
  • 64 位 CPU google/netstack 是这样的

Clash for Android 日前释出了预览版，bug有，但是总体上还是比较稳定的。作者也很勤奋，一天更新了多个版本。
