JavaWEB-（一）Maven搭建及配置

本文最后更新于：June 12, 2021 am

Maven是 Apache 下的一个纯 Java 开发的开源项目。基于项目对象模型（缩写：POM）概念，Maven利用一个中央信息片断能管理一个项目的构建、报告和文档等步骤。Maven 也可被用于构建和管理各种项目，例如 C#，Ruby，Scala 和其他语言编写的项目。Maven 曾是 Jakarta 项目的子项目，现为由 Apache 软件基金会主持的独立 Apache 项目。

Maven 是一个项目管理工具，可以对 Java 项目进行构建、依赖管理。

目录

1.下载Maven

官网：http://maven.apache.org/download.cgi

2.配置环境变量

  1. 在系统变量中新建两个变量：M2_HOME、MAVEN_HOME。

其中，
M2_HOME：路径为maven目录下一直至bin目录，如：D:\abc\abc\bin
MAVEN_HOME：路径为上面的路径不要bin目录的路径，如：D:\abc\abc

  1. 设置系统的path变量

在系统变量中，找到path变量，然后编辑，新建一个：%MAVEN_HOME%\bin ，然后再一步步确定确定即可。

  1. 检测是否配置成功

打开 CMD 窗口，输入 mvn -version，看见版本等相关消息后即表示成功配置。

4.在IDEA中使用Maven

  1. 在IDEA中创建一个Maven项目
  1. 在创建中，如遇到：Maven home directory、User settings file、Local repository，它们的填写如下。

Maven home directory：maven的目录
User settings file：maven目录下的conf下的settings.xml文件
Local repository：maven目录下的maven-repo文件夹仓库（这个仓库为自己所创的那个）。

创建完成后，IDEA将自动导入相应的包。

需要注意的是，每一次自己打开项目后都要看一下maven是否是自己的maven，而不是IDEA默认的maven。因为IDEA默认的maven是在C盘（如：C:\Users\DragonOne下有一个m2的文件夹，这个就是IDEA默认的）。

查看当前项目的maven：File -> settings -> Build,Execution,Deployment -> Build Tools -> Maven 中可以看见。
