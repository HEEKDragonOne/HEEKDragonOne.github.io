0.前提工作

需要下载对应主题的 jar 包。

1.导入

  1. 首先打开 file -> Import Settings，在弹出的窗口中再选择对应的主题 jar 包，再点击 OK 即可。

设置自定义的主题的字体大小需要注意顺序！
